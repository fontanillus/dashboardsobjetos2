El perfil de usuario

Destructuración de objetos en consola.

Ejercicio

Sacamos nombre, edad y pais del objeto usuario usando destructuración. 

const usuario = {
  nombre: "Carlos",
  edad: 28,
  pais: "España",
  profesion: "Desarrollador"
};

const { nombre, edad, pais } = usuario;
console.log(nombre, edad, pais);

Resultado esperado en consola

Carlos 28 España

Notas explicativas

  • usuario: objeto.
  • { }: destructuración de objeto.
  • nombre, edad, pais: variables nuevas.
  • console.log(): muestra el resultado en consola.

Cómo verlo

  1. 1. Abre

    Se carga el JS.

  2. 2. Consola

    F12 o Ctrl + Shift + J.

  3. 3. Resultado

    Carlos 28 España.

Nota

Usamos llaves { } porque estamos sacando datos de un objeto. Para arrays se usan corchetes [ ].
