El podio de una carrera

Destructuración de arrays en consola.

Ejercicio

Sacamos el primer, segundo y tercer corredor del array corredores. 

const corredores = ["Ana", "Luis", "María", "Pedro", "Sofía"];

const [oro, plata, bronce] = corredores;

console.log(oro, plata, bronce);

Resultado esperado en consola

Ana Luis María

Notas explicativas

  • corredores: array.
  • [ ]: destructuración de array.
  • oro: primer valor, Ana.
  • plata: segundo valor, Luis.
  • bronce: tercer valor, María.
  • Pedro y Sofía no se guardan.

Cómo verlo

  1. 1. Abre

    Se carga el JS.

  2. 2. Consola

    F12 o Ctrl + Shift + J.

  3. 3. Resultado

    Ana Luis María.

Luis

Oro

Nota

En arrays importa la posición. Por eso usamos corchetes [ ], no llaves { }.
