Configuración de una app

Destructuración con valores por defecto.

Ejercicio

Sacamos tema e idioma del objeto config. Si falta idioma, usamos "es". 

const config = {
  tema: "oscuro",
  // El idioma no viene aqui
};

const { tema, idioma = "es" } = config;

console.log(tema, idioma);

Resultado esperado en consola

oscuro es

Notas explicativas

  • config: objeto.
  • { }: destructuración de objeto.
  • tema: existe y vale oscuro.
  • idioma: no existe en config.
  • idioma = "es": valor por defecto.
  • console.log(): muestra el resultado.

Cómo verlo

  1. 1. Abre

    Se carga el JS.

  2. 2. Consola

    F12 o Ctrl + Shift + J.

  3. 3. Resultado

    oscuro es.

Tema

oscuro

Idioma

es

Nota

Si una propiedad no existe, el valor por defecto evita que la variable quede como undefined.
