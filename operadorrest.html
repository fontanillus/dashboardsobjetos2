Producto y operador rest

Destructuración de objetos con propiedades restantes.

Ejercicio

Sacamos id y precio del objeto producto. El resto de propiedades se agrupa en detalles. 

const producto = {
  id: "PROD-10",
  precio: 29.99,
  nombre: "Teclado Mecánico",
  marca: "Logitech",
  stock: 15
};

const { id, precio, ...detalles } = producto;

console.log(id, precio);
console.log(detalles);

Resultado esperado en consola

PROD-10 29.99

{ nombre: 'Teclado Mecánico', marca: 'Logitech', stock: 15 }

Notas explicativas

  • producto: objeto original.
  • { }: destructuración de objeto.
  • id: se guarda en variable.
  • precio: se guarda en variable.
  • ...detalles: agrupa lo restante.
  • console.log(): muestra el resultado.

Cómo verlo

  1. 1. Abre

    Se carga el JS.

  2. 2. Consola

    F12 o Ctrl + Shift + J.

  3. 3. Resultado

    id, precio y detalles.

ID

PROD-10

Precio

29.99

Rest

detalles

Nota

El operador ... recoge en un nuevo objeto las propiedades que no se han extraído antes.
