Datos anidados de un e-commerce

Destructuración de objetos dentro de objetos.

Ejercicio

Sacamos nombre del cliente y, dentro de direccion, extraemos ciudad y pais. 

const cliente = {
  id: "CLI-992",
  nombre: "Laura",
  direccion: {
    ciudad: "Barcelona",
    codigoPostal: "08001",
    pais: "España"
  }
};

const { nombre, direccion: { ciudad, pais } } = cliente;

console.log(nombre, ciudad, pais);

Resultado esperado en consola

Laura Barcelona España

Notas explicativas

  • cliente: objeto principal.
  • direccion: objeto anidado.
  • nombre: sale de cliente.
  • ciudad: sale de direccion.
  • pais: sale de direccion.
  • codigoPostal: no se extrae.

Cómo verlo

  1. 1. Abre

    Se carga el JS.

  2. 2. Consola

    F12 o Ctrl + Shift + J.

  3. 3. Resultado

    Laura Barcelona España.

Nota

Para entrar en un objeto anidado, escribimos su propiedad y abrimos otro bloque de llaves: direccion: { ciudad, pais }.
