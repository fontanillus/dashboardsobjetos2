Conflicto de variables

Destructuración con alias para renombrar.

Ejercicio

Sacamos title y salary del objeto vacante, pero los renombramos como titulo y salario. 

const vacante = {
  title: "Frontend Dev",
  salary: 45000,
  remote: true
};

const { title: titulo, salary: salario } = vacante;

console.log(titulo, salario);

Resultado esperado en consola

Frontend Dev 45000

Notas explicativas

  • vacante: objeto.
  • { }: destructuración de objeto.
  • title: titulo: crea titulo.
  • salary: salario: crea salario.
  • remote: no se usa.
  • console.log(): muestra el resultado.

Cómo verlo

  1. 1. Abre

    Se carga el JS.

  2. 2. Consola

    F12 o Ctrl + Shift + J.

  3. 3. Resultado

    Frontend Dev 45000.

title

Frontend Dev

salary

Nota

En title: titulo, title es la propiedad original y titulo es la nueva variable.
